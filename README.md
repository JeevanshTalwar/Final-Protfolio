# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML5** and **CSS3**. This website showcases my skills, projects, certifications, resume, and achievements as a Computer Science undergraduate and aspiring Software Engineer.

## 🚀 Live Demo

🔗 https://jeevansh-talwar.vercel.app

## 📌 Features

- Modern and responsive design
- Clean and minimal user interface
- About Me section
- Skills showcase
- Projects section
- Certifications section
- Beyond Academics section
- Downloadable Resume
- Contact information
- Built using only HTML & CSS

## 🛠️ Technologies Used

- HTML5
- CSS3
